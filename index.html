HTML

CSS

The 3 types of CSS

  • Inline: used to style a specific, single element directly in the HTML file.
  • Internal:used to style an entire stylesheet directly in the HTML file using a style element in the head element (meta code)
  • External: uses a seperate, linked file (linked in the HTML file's head) and is the most common/preferred method

Terms and Commands

  • Margin: space outside of an element, not visable on the page. Increasing value adds a buffer, 0 value indicates NO extra space around the element
  • Padding: space inside the element. Increasing value makes the entire element bigger.
  • * symbol: used to apply a rule to ALL the elements visible on the page
  • Link External CSS: in the head section of HTML file, after the final META but before the TITLE
  • Rel: specifies the relationship between current doc (ie the HTML file) and the linked doc (ie external CSS stylesheet)
  • Href: specifies the location (URL) of the external, linked resource and always starts with ./ (ie ./assets/style.css)

Git

  • git status: checks what branch you're currently working on
  • git add -A: stages changes to be committed
  • git commit -m "name/details of commit": commits changes
  • git pull/git pull (branch name): pulls from Github repo
  • git push/git push (branch name): pushes commits to Github repo
